```

REQUISITOS:

1. ADICIONAR EVENTO (NOME, DATA E VALOR DO INGRESSO);
2. ALTERAR EVENTO;
3. EXCLUIR EVENTO;
4. ORDENAR POR DATA;
5. CRIAR UMA ACTIVITY OU DIALOG QUE EXIBA A MÉDIA DOS PREÇOS.

OBS.: A EQUIPE DEVE SER COMPOSTA POR ATÉ 3 PESSOAS!
```
